# 本郷スタートアップご近所さん会 2017 秋（仮）

## 巻頭メッセージ（仮）

本郷には今たくさんのスタートアップや関係者が集まりつつあり、似たような事業のステージで、似たような悩みを抱える人が多く居るようです。

せっかく近くで働いてすれ違っているのですから、そんな人たちが少し集まって、お互いのリアルな課題や過去の知見を分かち合い、そして次の日に本郷でランチミーティングがもっと起これば全体に利すると考えており、そんな会を実験的に開催したいと思います。

### 参加者制限

翌日辺りに本郷周辺でランチ～ミーティングに行ける方に限ってお知らせいただけますと幸いです。

### 事例

運営メンバーの周りでは最近こんなことが起こっていました。こうした出来事をもっと多くして、本郷の周りでトランザクティブメモリ（誰が何を知っているかを知っている）を作るために今回の会を企画しています。

- どの派遣業者がいいのか、ふとした会話から話が弾んで選定できた
- 本郷のスタバでコーヒーを飲んでたら、知り合いが通りかかって他の人を紹介してくれて仕事が出来た
- 起業家と出会ったら、顧客を紹介されて自分の技術がぴったりとはまって起業した
- たまたま近くで会議をしていた友人の友人がエンジェル投資家になってくれた

## 目的

### イベントの目的

- （当日）知り合いの知り合い同士で会ったほうが良いのではな人たちがお互いを紹介し合う
- （翌日）気の合った人と次の月曜あたりに本郷辺りでランチミーティングに行く

= 忙しすぎて本郷に来れない人は、申し訳ありませんが参加をお断りさせていただくこともあります

### イベントの方針

- みんなが喋れるようなワークショップがメイン（成功者の話を聞くような会ではありません）
- 課題と解決策を共有する
- 大部屋は緩めのトーク
- 協力し合おうという気持ちになっている人が集まる
- 本郷や近隣をもっと良くしたいという思いのある人が集まる

### 従来と今回のイベントとの違い

|  | 従来 | 今回 |
|  ------ | ------ | ------ |
| スピーカー | 成功者や偉い人が壇上から話す | 参加した人全員が同じ目線で話す |
| 主な参加者 | 投資家＋事業会社＋スタートアップ経営者 | スタートアップの現場の従業員＋経営者 |
| 狙い | 投資＋会社間の協業 | 現場のノウハウ交換＋現場レベルでの協業 |
| 運営 | 営利企業 | 有志のボランティア |
| スポンサーの金額 | 数百万円（会社） | 数万円（一部の個人） |



### イベントが狙うビフォーアフター

|  イベント前 | イベント後 |
|  ------ | ------ |
|  プログラミングできれば効率化できるのに… | そうだ、あの人にちょっとだけ頼ってみよう |
|  この業界の顧客に会いたい…けど知らない… | あの人が知ってそう |

## セッションマスターの方へ

### セッションマスター

「このトピックならファシリテートしたい！情報共有したい！」という人（運営に関わってくれる人）を募集しています。

ただしファシリテーターも「今日会った人と、次の月曜に本郷でランチミーティングに行ける人」という方でお願いします。

やっていただきたいことは、セッショントピックのアイデア出しと、当日のファシリテート、参加者集めです。

### セッションの進め方

ワークショップはおおよそ以下のような進め方を想定しています。30 分しかないので効率よく進めてください。

| 時間 | 内容 |
| ------ | ------ |
| 1 分 | セッションの主旨説明とオーナーの自己紹介 |
| 1 分 | ポストイットとペンの配布確認（入り口で受け取っているか） |
| 1 分 | 現在の課題（黄色ポストイット）の書き出し（個人ワーク） |
| 1 分 | 対策としてやっていること（青色ポストイット）の書き出し |
| 3 分 | 隣に座っている人との自己紹介 & シェア |
| 4 分 | 全体のポストイットの張り出しとグルーピング |
| 2 分 | ポストイットへの投票（●シール貼り） |
| 1 分 | 全体でディスカッションする課題を 2 つ決定 |
| 3 分 | 1トピック目について別の隣の人とにディスカッション |
| 3 分 | 知見の全体への共有 (Think, Pair, Share) |
| 3 分 | 2トピック目について別の隣の人とにディスカッション |
| 3 分 | 知見の全体への共有 (Think, Pair, Share) |
| 4 分 | 雑談 ＆ お互い紹介タイム |

なお話し足りないときは、会場外で是非議論し続けてください。

### セッションルール

- あくまで上記の進め方は一例です。他に良い進め方があればお任せいたします。
- セッションマスターの方は隣に座った人同士がうまく喋れるように配慮してください。
- セッションマスターの方が一方的に喋るセッションは避けてください。そのような話を事後に伺った場合、次回以降の参加をご遠慮頂く場合がございます。
- セッション後、スタッフが張り出した紙を撮影して、Scrapbox にアップします。
