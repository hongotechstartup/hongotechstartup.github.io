# 本郷スタートアップご近所さん会 2017 秋（仮）

## 巻頭メッセージ（仮）

本郷には今たくさんのスタートアップや関係者が集まりつつあり、似たようなステージで共通の悩みを抱える人が多く居ます。

せっかく近くで働いてすれ違っているのですから、そんな人たちが少し集まって、善意でお互いのリアルな課題や過去の知見を分かち合い、そして次の日に本郷でランチミーティングがもっと起こらないかと考えていて、そんな会を実験的に開催したいと思います。

### 事例

運営メンバーの周りでは最近こんなことが起こっていました。こうした出来事をもっと多くして、本郷の周りでトランザクティブメモリ（誰が何を知っているかを知っている）を作るために今回の会を企画しています。

- どの派遣業者がいいのか、ふとした会話から話が弾んで選定できた
- 本郷のスタバでコーヒーを飲んでたら、知り合いが通りかかって他の人を紹介してくれて仕事が出来た
- 起業家と出会ったら、顧客を紹介されて自分の技術がぴったりとはまって起業した
- たまたま近くで会議をしていた友人の友人がエンジェル投資家になってくれた

興味のあるワークショップがない時間帯は、大部屋で PC 作業をしながら誰かの話をゆったり聞く（別に聞かなくてもそこにいていい）セッションを行います。

- あの Exit したスタートアップのクローズド話

#### 注意

一部セッションは、特定の職業の方は入れないような制限をつけます。クローズドの話をするための処置となります。ご容赦ください。

## セッションマスターの方へ

### セッションマスター

「このトピックならファシリテートしたい！情報共有したい！」という人（運営に関わってくれる人）を募集しています。

ただしファシリテーターも「今日会った人と、次の月曜に本郷でランチミーティングに行ける人」という方でお願いします。

やっていただきたいことは、セッショントピックのアイデア出しと、当日のファシリテート、参加者集めです。

### セッションの進め方

ワークショップはおおよそ以下のような進め方を想定しています。30 分しかないので効率よく進めてください。

| 時間 | 内容 |
| ------ | ------ |
| 1 分 | セッションの主旨説明とオーナーの自己紹介 |
| 1 分 | ポストイットとペンの配布確認（入り口で受け取っているか） |
| 1 分 | 現在の課題（黄色ポストイット）の書き出し（個人ワーク） |
| 1 分 | 対策としてやっていること（青色ポストイット）の書き出し |
| 4 分 | 隣に座っている人とのシェアとグルーピング |
| 4 分 | 全体のポストイットの張り出しとグルーピング |
| 2 分 | ポストイットへのシール貼り |
| 2 分 | 全体でディスカッションする課題を 2 つ決定 |
| 3 分 | 1トピック目について別の隣の人とにディスカッション |
| 3 分 | 知見の全体への共有 (Think, Pair, Share) |
| 3 分 | 2トピック目について別の隣の人とにディスカッション |
| 3 分 | 知見の全体への共有 (Think, Pair, Share) |

セッション後、スタッフが張り出した紙を撮影して、Scrapbox にアップ。

なお話し足りないときは、会場外で是非議論し続けてください。

### セッションルール

- 武勇伝や自慢話ではなく、議論や次の繋がりを生みそうなもの
- 完全招待制かつクローズドのイベントなので、本郷以外の方々にあまり広めないでください
- シークレットだからこそ喋られる内容をお勧めしています
