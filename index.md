# 本郷スタートアップご近所さん会 2017 秋（仮）

## 巻頭メッセージ（仮）

本郷には今たくさんのスタートアップや関係者が集まりつつあり、似たようなステージで共通の悩みを抱える人が多く居ます。

せっかく近くにいるのですから、そんな人たちが少し集まって、善意でお互いのリアルな課題や過去の知見を分かち合い、そして次の日に本郷でランチミーティングがもっと起こらないか、そんな会を実験的に開催したいと思います。

### 事例

運営メンバーの周りではこんなことが起こっています。

- どの派遣業者がいいのか、ふとした会話から話が弾んで選定できた
- 本郷のスタバでコーヒーを飲んでたら、知り合いが通りかかって他の人を紹介してくれて仕事が出来た
- 起業家と出会ったら、顧客を紹介されて自分の技術がぴったりとはまって起業した
- たまたま近くで会議をしていた友人の友人がエンジェル投資家になってくれた

こうした出来事をもっと多くして、本郷の周りでトランザクティブメモリを作るために今回の会を企画しました。

### 方針

- みんなが喋れるようなワークショップがメイン（成功者の話を聞くような会ではありません）
- 課題と解決策を共有する
- 協力し合おうという気持ちになっている人が集まる
- 本郷や近隣をもっと良くしたいという思いのある人が集まる

## セッションマスターの方へ

### セッションマスター

「このトピックで喋りたい！」という人（運営に関わってくれる人）を募集しています。

基準は「今日会った人と、次の月曜にランチミーティングに行ける人」です。

### セッションの進め方

ワークショップはおおよそ以下のような進め方を想定しています。30 分しかないので効率よく進めてください。

| 時間 | 内容 |
| ------ | ------ |
| 1 分 | セッションの主旨説明とオーナーの自己紹介 |
| 1 分 | ポストイットとペンの配布確認（入り口で受け取っているか） |
| 3 分 | 現在の課題の書き出し（個人ワーク） |
| 2 分 | ポストイットの張り出し |
| 3 分 | ポストイットのグルーピング |
| 2 分 | 課題へのシール貼り |
| 2 分 | ディスカッションする課題を 3 つぐらい決定 |
| 15 分 | 解決策を持っている人を中心にディスカッション |

話し足りないときは、会場外で是非議論し続けてください。

### セッションルール

- 武勇伝や自慢話ではなく、議論や次の繋がりを生みそうなもの
- 完全招待制かつクローズドのイベントなので、あまり広めないでください
- シークレットだからこそ喋られる内容をお勧めしています

## 招待する人の基準

すべて

- 単に人脈が広い人ではなく、過去に良い紹介をしてもらったことがある人
- つながりを持てば何か起こりそうな技術やスキルを持つ人
- 明日本郷でアクションを起こせる人（つまり、今忙しすぎるタイミングな人や偉すぎて本郷に居ない人は対象外）

たまたま今のタイミングが忙しいから、という人もいると思います。
